## Core Project: CodePilot

CodePilot 是当前 Portfolio 的核心展示项目。

项目定位：

```txt
GitHub Repository Understanding & AI Code Review
```

基本流程：

1. 输入 GitHub 仓库链接
2. 解析源码结构
3. 构建结构化上下文
4. 生成 Markdown 审查报告

CodePilot 仓库：

```txt
https://github.com/Strange-Men/CodePilot
```

## Development Rules

后续开发遵守：

- 每轮只做一个小任务
- 每轮必须 build 通过
- 不在未提交状态下继续开发
- 不编造项目、链接、数据和经历
- 不复制外部模板或未确认代码
- 重要版本通过 Git tag 记录
- 后续阶段性 commit message 必须带版本号，例如 `feat(v1.8.0-page-complete): compose final page`

详细规则见：

```txt
docs/15_PROJECT_CONTEXT.md
docs/16_DEV_RULES.md
docs/20_VERSION_HISTORY.md
```
