# Portfolio 原始想法

## 项目想解决什么问题

做一个面向实习投递的个人作品集网站，用来展示我的 AI 应用开发方向、CodePilot 项目、前端作品集开发能力和工程化工作流。

## 目标用户是谁

- 技术面试官
- HR
- 实习招聘方
- 查看我作品链接的人

## 用户为什么需要它

他们需要快速判断我是否具备 AI 应用开发、项目表达、前端实现、工程化迭代和持续交付能力。

## 它和普通个人主页有什么区别

普通个人主页只是展示信息。
这个作品集要体现：

- AI 应用开发方向
- CodePilot 项目能力
- Vibe Coding 工程化流程
- 可上线、可维护、可迭代的前端项目能力

## 最小可行版本是什么

第一版只做静态作品集页面，展示：

- Hero
- About
- CodePilot
- Portfolio 开发过程
- Skills
- Workflow
- Contact

## 这个项目最终希望证明什么能力

- AI 应用开发表达能力
- 项目包装与技术表达能力
- React + Vite 静态页面开发能力
- 设计基调控制能力
- Claude Code / Codex 小步迭代工作流能力
- Git 版本控制和可回滚开发能力
