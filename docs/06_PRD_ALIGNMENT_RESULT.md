## 4. 推荐默认策略

如果用户不再额外确认，最终 PRD 可采用以下默认策略：

### 4.1 Hero

中文为主，技术关键词保留英文。

方向：

```txt
2027 届人工智能本科生，聚焦 AI 应用开发 / Agent 开发 / Python 后端。
当前核心项目是 CodePilot，一个面向 GitHub 仓库理解与代码审查的 AI 应用项目。
```

### 4.2 语言

第一版使用中文为主，技术名词保留英文。

原因：

- 当前主要服务国内实习投递
- 中文表达更直接
- 技术关键词保留英文，便于面试官快速识别

### 4.3 视觉方向

第一版可以倾向暗色高级，但必须克制。

要求：

- 视觉服务内容
- 不做复杂动效
- 不做 ReactBits
- 不做 WebGL
- 不做 cursor spotlight
- 不让视觉抢走 CodePilot 的重点

### 4.4 CodePilot 展示深度

第一版展示：

- 项目定位
- 核心功能
- 技术栈
- 基本流程
- 当前进展
- 可讲技术点
- GitHub 链接

不展示过长版本迭代，不写虚假数据，不写未确认 Demo。

### 4.5 部署

默认后续优先考虑 Vercel。
但部署平台不在当前 PRD 对齐阶段决定。

## 5. 当前 MVP 最终倾向

第一版继续保持 4 个模块：

1. Hero
2. About
3. CodePilot
4. Contact

不新增 Skills、Workflow、Blog、Portfolio 开发过程。
